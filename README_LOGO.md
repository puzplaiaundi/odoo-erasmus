Colocar logo del módulo

1) Copia tu imagen a:

   gestion_erasmus/static/img/iconoplaiaundi.jpg

2) Accede a la imagen por URL:

   /gestion_erasmus/logo

   (Ruta pública, útil para probar o para usar en plantillas)

Notas:
- El fichero debe llamarse exactamente iconoplaiaundi.jpg. Si quieres otro nombre o tipo, avísame y lo parametrizamos.
- También puedes referenciarla directamente como asset estático: /gestion_erasmus/static/img/iconoplaiaundi.jpg
